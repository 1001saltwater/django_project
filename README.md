# AI Knowledge Platform

> **An AI-powered knowledge management platform built with Django, PostgreSQL and React.**
>
> **一个基于 Django、PostgreSQL 和 React 构建的 AI 知识管理平台。**

---

# 📖 Introduction | 项目简介

## English

AI Knowledge Platform is a full-stack web application that combines knowledge management with Large Language Models (LLMs).

Instead of being just a blogging system, the project aims to become an intelligent knowledge platform where users can organize articles, build personal knowledge bases, and interact with AI assistants through context-aware conversations.

This project is also a long-term engineering practice that covers backend development, database design, system architecture, performance optimization, deployment, and AI application integration.

## 中文

AI Knowledge Platform 是一个融合大语言模型（LLM）的全栈知识管理平台。

它不仅仅是一个博客系统，而是希望发展成为一个支持知识沉淀、内容管理和 AI 智能问答的平台。用户可以创建知识内容，并围绕知识与 AI 进行上下文相关的对话。

本项目也是一个长期持续迭代的工程实践，涵盖后端开发、数据库设计、系统架构、性能优化、部署以及 AI 应用开发等多个方向。
